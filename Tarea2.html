HERRAMIENTAS PRINCIPALES

HTML

Es el lenguaje que se usa para crear y estructurar páginas web.Define elementos como texto, imágenes, enlaces y botones dentro de un sitio web.

CSS

Es el lenguaje que se usa para dar estilo y diseño a las páginas web.Permite cambiar colores, tamaños, posiciones y apariencia de los elementos HTML.
